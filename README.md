# First Non-Repeating Character in a String (Python)

This project contains a Python program that finds the first
non-repeating character in a given string.

The program first counts the frequency of each character and then
checks the string in order to find the first character whose
frequency is exactly one.

This is a common beginner-level interview question.

---

## 📌 Problem Statement

Write a Python program that takes a string from the user and prints
the first character that does not repeat in the string.

If no such character exists, print a proper message.

---

## 🧪 Example

Input:
Enter a word: swiss

Output:
First non-repeating character is: w

---

Input:
Enter a word: aabbcc

Output:
No non-repeating character found
